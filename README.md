🎮 Plaza de Maipú 3D Game

Un pequeño juego en 3D inspirado en la Plaza de Maipú, desarrollado con HTML, CSS y JavaScript.

El proyecto busca recrear una experiencia simple e interactiva dentro de un entorno urbano en 3D, combinando diseño visual, movimiento y exploración en el navegador.

✨ Características
Mundo simple en 3D
Movimiento e interacción básica
Diseño inspirado en Plaza de Maipú
Efectos visuales y ambiente urbano
Proyecto liviano y rápido
Compatible con navegador web
🛠 Tecnologías usadas
HTML5
CSS3
JavaScript
Renderizado 3D en navegador
